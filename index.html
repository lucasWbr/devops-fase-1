Fase 1 - Integração GitHub com Amazon S3

Bem-vindo ao Projeto

Este é um projeto demonstrativo que integra GitHub com Amazon S3 para hospedagem de site estático.

Principais características:

  • Integração contínua com GitHub Actions
  • Hospedagem no Amazon S3
  • Deploy automático
  • Configuração de infraestrutura como código
